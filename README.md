# Digital Signal Processing Transforms App

A comprehensive web-based signal processing tool for computing various transforms and analyzing signals.

## Features

- **Z-Transform** - Compute along z-plane with ROC visualization
- **DFT/IDFT** - Discrete Fourier Transform and inverse
- **FFT/IFFT** - Fast Fourier Transform with DIT & DIF algorithms
- **DTFT/IDTFT** - Discrete-Time Fourier Transform
- **Butterfly Diagrams** - DIT and DIF FFT flow graphs
- **Multiple Signal Types** - Custom, exponential, sine, impulse, step
- **Real-time Visualization** - Magnitude & phase spectra, ROC diagrams

## Live Demo

[To View click here](https://nishit0072e.github.io/Digital-Signal-Processing-Transforms/)

## How to Use

1. Select a signal type or enter custom sequence
2. Choose a transform (Z, DFT, FFT, or DTFT)
3. Click compute button
4. View results in charts and output box

## Installation (Local)
```bash
git clone https://github.com/nishit0072e/Digital-Signal-Processing-Transforms.git
cd Digital-Signal-Processing-Transforms
# Open index.html in your browser
```

## Technology Stack

- Vanilla JavaScript (no frameworks)
- Chart.js 3.9.1 for visualizations
- HTML5 Canvas for butterfly diagrams
- Pure CSS for styling
